## note

Ask each plane: is there a direction multiplication only *slides* along, never
turns? — a **rail**. The rail count is the real roots of `t² = p`:

- `p > 0` → **2** rails (split)
- `p = 0` → **1** (dual)
- `p < 0` → **0** (complex)

One count; three faces.

## full

Rails are real eigenvectors of "multiply by `j`," eigenvalues solving `t² = p`. Two
real roots → diagonalizable → `ℝ×ℝ`; one → a defective shear; none → a genuine
rotation with no real invariant line. The same number is the unit-curve type and the
orbit shape. [[CX]] [[DU]] [[SP]]
