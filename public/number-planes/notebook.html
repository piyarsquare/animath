I · What do we want in a plane number?

Two coordinates — one real, and one along j, the second direction.
z = x + y·j
We can add z and a second number, w = a + b·j — the parts add on their own; nothing to decide.
(x + y·j) + (a + b·j) = (x+a) + (y+b)·j
We can multiply — respecting the distributive property, and so:
z·w = x·a + (x·b + y·a)·j + y·b·j²
Every symbol is arithmetic we already own — except j². Until we say what it is, the last term has nowhere to land.
One number. That is the whole freedom of the plane.
j² = −1.00
(x + y·j)(a + b·j)
Hmm? — that last tile, y·b·j²: real part, or j part? j² is itself a plane number — if it is real, the tile joins x·a. It can be:

p can be any real number. Infinitely many planes, then?

Try measuring j with a different ruler:

ĵ = j / s   ⇒   ĵ² = p / s²

Each p can be rescaled by any s² — touch each number and let it settle.

every p is a plane — tap one
Three planes. One dial. Only the sign matters — and now you know why the middle sticks: nothing can rescale its way across zero.

Before asking how they differ — what do they share? Three things, and they are not small:
They all treat + the same.
the same tip-to-tail parallelogram, in every plane
They all treat real numbers the same.
on the shared axis, (x)(u) = xu — common ground in every world
They all treat ×1 as doing nothing.
the identity is the identity, wherever the dial points
So all the difference lives in one act: multiplying by a number off the real axis.
And one square shows it all: the corner 1+j is just (1,0) + (0,1) — and since × distributes over +, it must land at w + w·j. Where the two basis corners go decides everything.
Each plane has a mirror: flip the j-part.
z̄ = x − y·j
Multiply a number by its mirror — the same move as before:
z·z̄ = x² − xy·j + xy·j − y²·j² = x² − p·y²
the j-part cancels; out of every z comes a plain real number. Call it N(z).
And it respects multiplication:
N(z·w) = N(z)·N(w)
▸ check it N(z)N(w) = (z·z̄)(w·w̄) = (zw)·(z̄w̄) — and the mirror of a product is the product of the mirrors, since flipping j respects + and ×.
Each world carries its own size — where N holds constant are its circles.

×w is a linear operator

the characteristic equation is t² = p in disguise (λ = a ± b·√p) — rails are its real eigenvectors.

Where do 1 and j land? The columns are exactly that: w·1 = a + b·j is the first column (a, b); w·j = a·j + b·j² = p·b + a·j is the second (p·b, a).

j² appears only where j meets j — so exactly one entry, p·b, can feel the dial.

Same fact on the stage: the square's bottom edge is 0 → 1, so its image is 0 → w in every plane. Only the j-edge bends.

And the far corner rides for free: 1+j = (1,0) + (0,1), and × distributes over +, so it lands at w + w·j — the whole square follows from where the two basis corners go.
