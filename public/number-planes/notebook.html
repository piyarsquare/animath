Step to two coordinates. Addition rides along unchanged. Multiplication collapses to a single number: what j² equals.

Call it p. Turn it, and only the sign matters.

Only two demands: distribute over +, and keep a 1. On the line they force everything. Here they leave exactly one number loose.

×w is a linear operator

the characteristic equation is t² = p in disguise (λ = a ± b·√p) — rails are its real eigenvectors.

Where do 1 and j land? The columns are exactly that: w·1 = a + b·j is the first column (a, b); w·j = a·j + b·j² = p·b + a·j is the second (p·b, a).

j² appears only where j meets j — so exactly one entry, p·b, can feel the dial. The rest of the matrix doesn't know which plane it lives in.

Same fact on the stage: the square's bottom edge is the segment 0 → 1, so its image is 0 → w in every plane. Only the j-edge bends when p turns.

Margin

Let z = x + jy, j² = p. Pair z with its conjugate z̄ = x − jy:  z·z̄ = x² − p·y² ≡ N(z).
N(zw) = N(z)·N(w) — the invariant every plane respects. Its level sets are the stage’s geometry; its null set is where division dies.
